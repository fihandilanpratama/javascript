Timing Function Pada javascript

setTimeout() : menjalankan program/baris kode setelah menunggu sejumlah waktu tertentu. gunakan clearTimeout untuk memeberhentikan sebelum waktu yang ditetapkan.

setInterval() : menjalankan program/baris kode secara berulang-ulang.

keduanya merupakan bagian dari object window tapi karena window itu global maka tidak perlu manambahkan keyword window di depannya.
